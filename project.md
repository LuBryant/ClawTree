ClawTree
ClawTree（树爪智动）—— AI Agent 驱动的 Web3+AI 高校活动杀手级协作引擎
一句话简介（推荐直接用）： ClawTree 是基于 OpenClaw 开源 AI Agent 构建的 Genesis 跨界杀手级平台，专为 Web3+AI 生态（如大树财经高校行与 AI 高效活动）打造“发现-聚合-智能外联-趋势形成”闭环。通过多 Agent 协同（事件采集、智能外联、回复解析、AI客服），助力组织效率提升 70%+，规模化人才生态并形成活动潮流。高度融合 AI Agent 与 Web3 跨界创新，契合本次赛题特别鼓励方向。
---
详细描述
1. 项目背景与痛点 大树财经作为 Web3+AI 媒体与活动品牌，正大力推动全球高校行、AI 高效型黑客松及嘉宾分享，深度连接产业、资本与高校人才生态（已支持 HTX Genesis Hackathon 等）。
然而，手动模式难以规模化：高校 AI/Web3 活动信息分散，人工收集效率低；外联高校耗时耗力；缺乏趋势洞察，难形成潮流；高校方对平台了解不足，沟通门槛高；混合线上线下活动协调困难。
2. 产品愿景与解决方案 愿景：打造 AI 驱动的高校 Web3+AI 活动生态操作系统，让大树财经成为最聪明、高效的连接器与趋势引领者。
系统分为两大端：
• 普通用户端（高校学生、老师、参与者）：登录后浏览大树财经参与的活动、照片回顾、演讲主题与内容总结，通过 AI 大树财经客服 Agent 24/7 咨询合作模式、活动流程、资源支持等，降低沟通门槛。
• 管理端（大树财经团队）：功能强大，包含以下核心 Agent 技能：
• 事件采集技能：自动检索并结构化聚合高校 AI/Web3 活动（来源可溯），收集有联系方式的活动，便于后续联系学校方合作举办 AI/Web3 活动。 • 智能外联技能：生成个性化邀请邮件（价值主张 + 混合线上选项），支持批量限速发送至官方活动邮箱。 • 回复智能解析技能：实时监控收件箱，LLM 结构化输出（意图分类、关键联系人、建议行动），自动通知团队。 • AI 大树财经客服 Agent：同时服务普通用户端，解答疑问并转化意向。 • Web Dashboard（Vue3 + FastAPI）：事件浏览器 + 外联管道看板 + 趋势洞察 + 活动回顾（照片、演讲内容），支持混合活动协调。
MVP 目标（黑客松可快速实现）：聚合 50+ 活动、多 Agent 协同演示、用户端 + 管理端基本界面。短期试点覆盖 100+ 高校，正向回复率 >15%，节省团队 10-20 小时/次活动，管道增长 3 倍+。
3. 与赛题匹配及创新点 • Genesis 跨界杀手级应用：多 Agent 协同（采集、外联、客服、解析）赋能 Web3+AI 真实生态。 • AI × Web3 融合（特别鼓励）：服务 Web3 人才生态，未来可扩展链上事件登记、$HTX 激励模型。 • 技术亮点：OpenClaw 多技能编排 + Vibe Coding。 • 落地价值：真实用户（大树财经）+ 双端设计 + WAIC 展示潜力，最终开源核心技能。
4. 团队与执行力 开发者有 OpenClaw 实战经验、Vue3 建站（校友系统）、高校资源与 Web3 项目背景，执行力强，可快速迭代。
